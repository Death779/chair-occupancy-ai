# Проект для производственной практики: Анализ заполненности зала

Веб-приложение на Flask для детекции людей и стульев на изображении с использованием модели YOLOv8.

## Требования
- Python 3.9+
- Установленные зависимости из `requirements.txt`

## Установка и запуск (Локально)
1. Откройте терминал в папке `python_project`.
2. Создайте виртуальное окружение (опционально):
   ```bash
   python -m venv venv
   source venv/bin/activate  # Для Linux/Mac
   venv\Scripts\activate     # Для Windows
   ```
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Запустите приложение:
   ```bash
   flask run
   ```
   Или просто:
   ```bash
   python app.py
   ```
5. Откройте браузер по адресу: `http://127.0.0.1:5000`

## Структура
- `app.py` — основной файл сервера Flask и логика обработки YOLOv8.
- `requirements.txt` — зависимости проекта.
- `templates/` — HTML-шаблоны веб-интерфейса.
- `static/` — создается автоматически, содержит загруженные оригиналы и обработанные результаты.
- `database.db` — база данных SQLite (создается при первом запуске).
